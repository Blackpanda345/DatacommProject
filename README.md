# Data Communication Assignment Portfolio

A comprehensive, professional web portfolio showcasing Data Communication studies, concepts, and real-world applications.

## 📋 Project Overview

This project is a complete data communication assignment presentation that demonstrates understanding of:
- Communication fundamentals and core concepts
- Historical evolution from telegraph to modern protocols
- Network technologies and protocols (TCP/IP, HTTP, Ethernet, WiFi, 5G)
- Real-world applications and practical implementations
- Future trends in data communication

## 🚀 Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, Edge)
- Git (for version control)
- GitHub account

### Installation & Setup

1. **Clone the Repository**
```bash
git clone https://github.com/yourusername/data-communication-assignment.git
cd data-communication-assignment
```

2. **Open the Website**
   - Option A: Double-click `index.html` to open in your default browser
   - Option B: Right-click `index.html` → Open With → Choose your browser
   - Option C: Use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

3. **Customize Your Information**
   - Edit the profile section with your actual name and student ID
   - Replace the placeholder avatar with your own image
   - Update the bio section with your information
   - Modify links to point to your actual assignment files

## 📚 Content Sections

### Section 1: Overview
Introduces the five key components of data communication:
- Message
- Sender
- Receiver
- Transmission Medium
- Protocol

### Section 2: Concepts
Covers transmission modes and signal types:
- Simplex, Half-Duplex, Full-Duplex
- Analog vs Digital signals
- Modulation principles

### Section 3: History
Features a detailed **telegraph system diagram** showing:
- Telegraph key (sender)
- Battery (power source)
- Transmission line (wires)
- Electromagnet (receiver)
- Signal flow and Morse code

Timeline from 1830s to present day.

### Section 4: Technologies
